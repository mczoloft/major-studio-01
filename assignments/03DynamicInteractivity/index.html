ECI vs GDP per Capita Changes vs

We now try to look into other variables against the ECI; the GDP per capita change from last year, and the GINI. All of them are from 2016.

How this work was done

The slopegraph has 3 steps; 1st, the ECI; 2nd, GDP changes per capita; 3rd, GINI. The vertical position of the corners of each line was normalized to the axis height. The visualization was rendered with D3.
